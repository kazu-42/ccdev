# Product Overview

Claude Code Web セルフホスト版 - ブラウザベースのAI対話・コード実行環境

## Core Capabilities

1. **AI対話**: Anthropic Claude APIを利用したチャットベースのコーディング支援
2. **サンドボックス実行**: Cloudflare Sandbox上での安全なコード実行（JS/TS/Python）
3. **PWA対応**: スマートフォンからもアプリライクに利用可能
4. **セルフホスト**: 自分のCloudflareアカウントで完結するデプロイ

## Target Use Cases

- **個人開発者**: 外出先やモバイルから軽量にコード実行・AI相談
- **チーム開発**: 共有ツールとしてのAIコーディング環境
- **学習用途**: 安全なサンドボックスでのコード実験

## Value Proposition

- **完全セルフホスト**: データもAPIキーも自分の管理下
- **Cloudflareエッジ**: グローバルで低レイテンシなアクセス
- **PWAファースト**: スマホでもデスクトップ並みの体験
- **Tool Use対応**: AIが自律的にコード実行・ファイル操作

## MVP機能

1. チャットUI（Markdown・コードブロック対応）
2. AIストリーミングレスポンス
3. JavaScript/TypeScript実行
4. Python実行（Pyodide経由）
5. PWA（オフライン基本UI・ホーム画面追加）
