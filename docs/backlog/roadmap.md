## Tier 1: 必須機能（競争参加の前提条件）

### 1. Git統合
**優先度**: P0
**工数**: 2-3週間
**ステータス**: 未着手

#### 要件
- GitHub OAuth認証
- リポジトリ clone/pull/push
- ブランチ操作（checkout, create, merge）
- コミット履歴表示
- diff表示

#### 技術要件
- GitHub App / OAuth App登録
- Worker内でのgitコマンド実行
- 認証トークン管理（KV or D1）

---

### 2. ファイル永続化
**優先度**: P0
**工数**: 1-2週間
**ステータス**: 未着手

#### 要件
- Sandbox再起動後もファイルが残る
- プロジェクト単位でのファイル保存
- ファイルバージョニング（オプション）

#### 技術要件
- R2 or D1でのファイル保存
- Sandbox起動時のファイル復元
- 差分同期（変更分のみ保存）

---

### 3. パッケージ管理
**優先度**: P0
**工数**: 1-2週間
**ステータス**: 未着手

#### 要件
- npm/yarn/pnpm install UI
- pip install UI
- package.json / requirements.txt 解析
- インストール進捗表示

#### 技術要件
- Sandbox内でのパッケージマネージャー実行
- node_modules / site-packages永続化
- キャッシュ戦略
